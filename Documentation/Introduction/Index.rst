What does it do?
================

This extension provides Polish translations for the country, country subdivision, currency, language, and territory records of static_info_tables. For this purpose, it adds one or two fields to those tables and fills them with Polish data, if available.
Record view of the "Polish" language record with new field "Name (PL)" filled with the Polish translation.
